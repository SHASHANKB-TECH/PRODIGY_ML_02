# K-Means Clustering

This project demonstrates K-Means Clustering using Python and Scikit-learn for data clustering and visualization.

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib
- Scikit-learn

## Features
- Data preprocessing
- K-Means clustering
- Cluster visualization
- Data analysis

## Files
- kmeans_clustering.py → Main source code
- kmeans_clustering.png → Output graph/result
- requirements.txt → Required libraries

## How to Run
```bash
pip install -r requirements.txt
python kmeans_clustering.py
```
